# TV-TENNIS
Симулятор настольного тенниса (Pong) для CdM-8. Вся игра на «железе» в Logisim + ассемблер для таблицы рекордов.

### Аппаратная часть:
* Физика мяча на алгоритме Брезенхэма (углы отскока зависят от зоны попадания в ракетку)
* Предиктивный ИИ - бот считает точку будущего удара с учётом рикошетов
* Механика freeze - при зажатой кнопке в момент удара мяч «прилипает» к ракетке
* Динамическое ускорение игры каждые 5 отбиваний
* Разделение тактовых частот для игры и процессора
* Аппаратная валидация имени с клавиатуры
### Программная часть:
* Таблица рекордов (топ-5) с косвенной адресацией - сортируются указатели, а не сами 9-байтные блоки
* Обработчик прерывания counter_handler по окончании игры
* Memory-Mapped I/O для общения с периферией
* BCD-формат счёта с декодированием в ASCII

Управление:<br>
Set_Name - ввод имени (до 8 символов)<br>
L / R - движение ракетки<br>
Reset - сброс матча<br>
Игра идёт до 5 промахов<br>
